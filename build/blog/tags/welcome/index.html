SpaceAPI

SpaceAPI is an API to retrieve Information Regarding the Planets. A complete list of all crewed spaceflights, with details on each mission’s accomplishments and crew, is available in the section of the titular "SpaceAPI". The data and images are used without claim of ownership and belong to their respective owners. This API is open source and uses a BSD license.
